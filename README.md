## 夢なき者に成功なし

```
夢なき者に理想なし、
理想なき者に計画なし、
計画なき者に実行なし、
実行なき者に成功なし。
故に、夢なき者に成功なし。
```

## 教育勅語
```
朕（ちん）惟ふ（おもう）に、
我が皇祖皇宗（こうそこうそう）、
国を肇（はじ）むること宏遠（こうえん）に、
徳を樹（た）つること深厚なり。

我が臣民、克（よ）く忠（ちゅう）に克（よ）く孝（こう）に、
億兆（おくちょう）心を一（いつ）にして、
世世（よよ）　厥（そ）の美を済（な）せるは、
此（こ）れ　我が国体の精華（せいか）にして、
教育の淵源（えんげん）　
亦（また）実（じつ）に此（ここ）に存（そん）す。

爾（なんじ）臣民、
父母に孝（こう）に兄弟（けいてい）に友に、
夫婦相（あい）和（わ）し
朋友（ほうゆう）相（あい）信じ、
恭倹（きょうけん）己れ（おのれ）を持（じ）し

博愛（はくあい）衆（しゅう）に及（およ）ぼし、
学を修め業（ぎょう）を習ひ（ならい）、

以て智能（ちのう）を啓発（けいはつ）し
徳器（とっき）を成就し、
進（すすん）で公益を広め　世務（せいむ）を開き、

常に国憲を重じ（おもんじ）国法に遵ひ（したがい）、
一旦（いったん）緩急（かんきゅう）あれば
義勇公（こう）に奉（ほう）じ、

以て　天壤（てんじょう）無窮（むきゅう）の皇運を扶翼（ふよく）すべし。
是（かく）の如きは、独り朕が忠良（ちゅうりょう）の臣民たるのみならず、
又以て爾（なんじ）祖先の遺風を顕彰するに足らん。

斯（こ）の道は、実に我が皇祖皇宗の遺訓（いくん）にして、
子孫臣民の倶（とも）に遵守（じゅんしゅ）すべき所、
之を古今に通じて謬（あやま）らず、
之を中外（ちゅうがい）に施して悖（もと）らず、

朕（ちん）　爾（なんじ）臣民と倶（とも）に、
拳拳服膺（けんけんふくよう）して、
咸（みな）　其（その）徳を一（いつ）にせんことを庶（こい）幾ふ（ねがう）。

明治二十三年十月三十日
御名（ぎょめい）　御璽（ぎょじ）

```
